## 🚀 About Us
We are **TechGenZ**, a collective of Faculty of Applied Science students (combining the Departments of Computer Science and Physical Science) at Trincomalee Campus. We specialize in engineering next-generation hardware and software solutions, rapid prototyping, IoT, AI, and embedded systems.

Our mission is driven by student-led innovation, multidisciplinary collaboration, and real-world problem solving.

## ✨ Features
- **Trophy Wall**: A showcase of our competition milestones, projects, and hackathon achievements (e.g., YarlGeek Innovation Festival, SLIoT Challenge, Cre8x).
- **The Squad**: Meet the minds behind the innovation—our dedicated team members and their skill sets.
- **Media Album**: A visual gallery of our journey, late-night debugging, and hardware prototypes.
- **Contact Form**: Direct channel to reach out to our team captain for collaborations and mentorship.

## 🛠 Tech Stack
This project is built using modern web technologies to ensure a fast, responsive, and highly polished user experience.
- **Framework**: [React 19](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 💻 Getting Started

Follow these steps to run the project locally.

### Prerequisites
- Node.js (v18+ recommended)
- npm

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Dhawanharan/techgenz-website.git
   cd techgenz-website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000` (or the port specified by Vite).

## 📦 Building for Production
To generate a production-ready static bundle:
```bash
npm run build
```
The optimized files will be outputted to the `dist` directory, ready to be deployed to static hosting platforms like Vercel, Netlify, or GitHub Pages.
